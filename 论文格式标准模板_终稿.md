# EduPaper 论文格式标准模板（终稿）

> 本文档是 EduPaper 项目的论文格式标准模板，基于教育类期刊论文格式规范和 GB/T 7714-2015 参考文献著录规则编制。后续所有论文生成均以此为格式基准。

---

## 一、格式说明

### 1.1 页面设置

| 项目 | 规格 |
|------|------|
| 纸张 | A4（210×297mm） |
| 上下边距 | 2.54cm |
| 左右边距 | 3.17cm |
| 页眉距边界 | 1.5cm |
| 页脚距边界 | 1.75cm |
| 页码 | 页脚居中，阿拉伯数字 |

### 1.2 字体字号

| 要素 | 字体 | 字号 | 说明 |
|------|------|------|------|
| 论文标题 | 黑体 | 二号（22pt） | 居中，加粗 |
| 一级标题（一、） | 黑体 | 三号（16pt） | 段前段后各0.5行 |
| 二级标题（（一）） | 黑体 | 四号（14pt） | 左对齐 |
| 三级标题（1. ） | 宋体加粗 | 小四（12pt） | 左对齐 |
| 正文 | 宋体 | 小四（12pt） | 首行缩进2字符 |
| 摘要/关键词 | 宋体 | 小四（12pt） | "摘要""关键词"加粗 |
| 参考文献 | 宋体 | 五号（10.5pt） | 悬挂缩进2字符 |
| 表格内文字 | 宋体 | 五号（10.5pt） | 单倍行距 |
| 页眉/页脚 | 宋体 | 小五（9pt） | 居中 |

### 1.3 行距

- 正文：1.5倍行距
- 参考文献：1.5倍行距
- 表格内：单倍行距
- 段间不留空行

### 1.4 标题层级

```
一、一级标题              ← 黑体三号
（一）二级标题             ← 黑体四号
1. 三级标题               ← 宋体小四加粗
（1）四级标题             ← 正文内联
```

### 1.5 参考文献格式（GB/T 7714-2015）

**期刊 [J]：** `作者. 题名[J]. 刊名, 年(期): 起止页码.`
**专著 [M]：** `作者. 书名[M]. 出版地: 出版者, 年: 页码.`
**标准 [S]：** `起草单位. 标准名称[S]. 出版地: 出版者, 年.`
**政策文件 [A/OL]：** `发布机构. 文件名[A/OL]. (发布日期)[引用日期]. URL.`

> 注意：期刊格式为 `年(期):`，不写"第"和"期"字。如 `2023(3): 45-48`。

## 二、示例论文

以下是一篇完整的教学案例论文示例，展示各部分的正确格式。

---

# 种植区域测量中的长方形面积量感建构——以"雅趣园"种植区为例

**摘要：** 几何量感是小学数学核心素养的重要内容，但当前面积教学普遍存在重计算轻体验、脱离真实情境的问题。本研究以岑溪市第五小学"雅趣园"劳动基地为场域，在人教版三年级下册"长方形面积"教学中融入种植区域测量任务，引导学生在真实劳动情境中建构面积量感。教学实施后，实验班后测均分78.5分，较前测提升26.2分，显著高于对照班的9.8分提升。"估测—实测—反思"的教学路径促进了量感从"数"向"量"的转化，劳动实践为面积量感建构提供了不可替代的真实体验。

**关键词：** 量感；长方形面积；项目化学习；劳动教育；小学数学

## 一、问题提出

《义务教育数学课程标准（2022年版）》将量感列为小学数学核心素养的重要表现，明确指出量感是对事物的可测量属性及大小关系的直观感知[1]。在图形与几何领域，几何量感具体表现为对长度、面积、角度等几何量的感性体验和测量判断能力。当前小学数学量感教学却面临诸多困境。曹培英指出，教师对量感认识不足、对度量意义的理解不够深入、量感体验重视不够，导致理论与实践脱节[2]。

## 二、教学设计与实施

### （一）教学设计概要

本课以人教版小学数学三年级下册第五单元《面积》P66-67为教材依据[3]，依托"雅趣园"种植区开展项目化学习。教学目标定位于三点：数学层面，理解长方形面积=长×宽的计算方法；劳动层面，在种植区进行实地测量，掌握卷尺测量技能；量感层面，通过估测与实测对比，建立1平方米的面积量感。

### （二）教学实施过程

**1. 创设情境，提出问题**

教师带学生到种植箱旁，指着箱面问："这个种植箱的面是什么形状？它的面积大概有多大？"学生观察后尝试估测。学生1脱口而出"10平方米！"教师随即出示1平方米纸板放在种植箱旁，学生2惊呼"1平方米好大啊"，随即修正估测为3平方米。

**2. 动手探究，推导公式**

教师引导学生思考"用面积单位去铺太麻烦，有没有更快的办法"，学生分组用卷尺测量种植箱的长和宽。第三组在测量中出现了典型错误：学生5将2.6米读成"26米"，经学生6提醒后修正为2.6米。

### （三）教学效果数据

为检验教学效果，对实验班（120人）与对照班（120人）进行前后测。

表1 实验班与对照班前后测对比

| 班级 | 样本数 | 前测均分 | 后测均分 | 提升幅度 |
|------|--------|----------|----------|----------|
| 实验班 | 120人 | 52.3 | 78.5 | 26.2 |
| 对照班 | 120人 | 51.8 | 61.6 | 9.8 |

实验班提升幅度（26.2分）显著高于对照班（9.8分），差值16.4分。

## 三、教学反思与分析

**成功之处。** 真实劳动情境激发了学习兴趣，学生在雅趣园测量种植箱时表现出高度参与热情。估测—实测对比的设计促进了量感发展，学生通过1平方米参照物逐步建立了面积估测能力。

**不足之处。** 部分小组估测能力较弱，第四组未使用参照物直接猜测，误差达0.88平方米，说明量感培养需要更多估测练习。

## 四、结语

本研究在"雅趣园"劳动基地的真实情境中，以种植区域测量为载体，探索了长方形面积量感建构的教学路径。"估测—实测—反思"的设计促进了量感从"数"向"量"的转化，劳动实践为面积量感建构提供了不可替代的体验基础。实验班后测提升26.2分，显著高于对照班，验证了项目化劳动实践在量感培养中的价值。

## 三、格式要点速查

### 常见错误修正表

| 错误写法 | 正确写法 | 说明 |
|---------|---------|------|
| `2023, 第3期: 45-48` | `2023(3): 45-48` | 期刊期数不加"第""期"字 |
| `2023, 3期: 45-48` | `2023(3): 45-48` | 期数用括号包裹 |
| 全角标点 `．，：` | 半角标点 `. , :` | 参考文献用半角 |
| 3个作者用"等" | 4个以上才用"等" | "等"仅用于4+作者 |
| 参考文献按字母排序 | 按引用顺序排列 | GB/T 7714用顺序编码制 |
| 正文首行不缩进 | 首行缩进2字符 | 中文论文标准 |
| 单倍行距 | 1.5倍行距 | 教育类论文标准 |
| ## 一、标题（Markdown） | 导出后为黑体三号 | 见导出对应表 |
