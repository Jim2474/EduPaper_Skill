# 论文格式规范

基于教育类期刊论文格式标准 + GB/T 7714-2015 参考文献著录规则。
本文档是 EduPaper 论文格式的权威规范，paper-writer 和 edupaper-exporter 均须遵循。

## 三、行距与段落

| 要素 | 行距 | 段前段后 |
|------|------|---------|
| 论文标题 | 单倍行距 | 段后1行 |
| 正文 | 1.5倍行距 | 段前0行，段后0行 |
| 一级标题 | 1.5倍行距 | 段前0.5行，段后0.5行 |
| 二级标题 | 1.5倍行距 | 段前0.5行，段后0行 |
| 三级标题 | 1.5倍行距 | 段前0行，段后0行 |
| 参考文献 | 1.5倍行距 | 段前0.5行 |

### 正文段落格式

- 首行缩进2字符（约0.74cm，即2个汉字宽度）
- 段落之间不留空行
- 不使用空格代替缩进

## 四、标题层级

教育类论文采用中文数字编号体系，最多四级：

```
一、一级标题              ← 黑体三号，段前段后各0.5行
（一）二级标题             ← 黑体四号
1. 三级标题               ← 宋体小四加粗
（1）四级标题             ← 宋体小四，正文内联
```

**编号规则：**
- 一级：一、二、三、四、五、六、七、八、九、十
- 二级：（一）（二）（三）…
- 三级：1. 2. 3. …（注意：阿拉伯数字后用点号"."，不用顿号"、"）
- 四级：（1）（2）（3）…（正文内联，不单独成行）

**示例：**
```
一、问题提出

（一）研究背景

1. 量感教学的困境

（1）教师认知不足
```

> EduPaper 教学案例论文通常使用两级标题（一、+（一）），最多三级。

## 五、摘要与关键词

### 摘要格式

- "摘要："二字加粗（黑体小四），后接摘要内容（宋体小四）
- 摘要内容200-300字，单段
- 结构：背景问题（1句）→ 研究方法与范围（1句）→ 主要发现（1-2句，含数据）→ 结论（1句）
- 不使用"本文""笔者"等第一人称

### 关键词格式

- "关键词："三字加粗（黑体小四），后接关键词（宋体小四）
- 3-5个关键词，用分号"；"分隔
- 最后一个关键词后不加标点

**示例：**
```
摘要：几何量感是小学数学核心素养的重要内容……促进了量感从"数"向"量"的转化。

关键词：量感；长方形面积；项目化学习；劳动教育；小学数学
```

## 六、表格格式

- 表格标题在表格上方，格式："表1 ……"（黑体五号，居中）
- 表格使用三线表（上下边线+表头下线，无竖线）
- 表格内文字：宋体五号，单倍行距
- 表格编号按全文顺序：表1、表2、表3……

**三线表示例：**
```
表1 实验班与对照班前后测对比

─────────────────────────────────
班级    样本数   前测均分   后测均分   提升幅度
─────────────────────────────────
实验班   120人    52.3      78.5      26.2
对照班   120人    51.8      61.6       9.8
─────────────────────────────────
```

Markdown 中用 pipe table 即可，导出时自动转为三线表。

## 七、参考文献格式（GB/T 7714-2015）

### 正文引用

- 顺序编码制：引用处标注 `[序号]`，如"研究表明[3]，……"
- 序号按文中首次出现顺序编号
- 多处引用同一文献，使用相同序号

### 文末参考文献列表

- 标题"参考文献"使用黑体三号，居中
- 各条目使用宋体五号，1.5倍行距
- 悬挂缩进：首行顶格，续行缩进2字符
- 按正文引用顺序排列，不按字母排序

### 各文献类型格式

**期刊 [J]：**
```
[序号] 作者. 题名[J]. 刊名, 年(期): 起止页码.
[序号] 作者. 题名[J]. 刊名, 年, 卷(期): 起止页码.
```

**专著/教材 [M]：**
```
[序号] 作者. 书名[M]. 版次. 出版地: 出版者, 年: 页码.
```

**标准 [S]：**
```
[序号] 起草单位. 标准名称[S]. 出版地: 出版者, 年.
```

**政策文件（在线）[A/OL]：**
```
[序号] 发布机构. 文件名[A/OL]. (发布日期)[引用日期]. URL.
```

**学位论文 [D]：**
```
[序号] 作者. 题名[D]. 保存地: 保存单位, 年.
```

### 常见错误

- ❌ `2023, 第3期: 45-48` → ✅ `2023(3): 45-48`
- ❌ `2023, 3期: 45-48` → ✅ `2023(3): 45-48`
- ❌ 使用全角标点 `．，：` → ✅ 使用半角标点 `. , :`
- ❌ 3个及以下作者用"等" → ✅ 仅4个以上作者用"等"
- ❌ 参考文献按字母排序 → ✅ 按正文引用顺序排列

## 八、数字与单位

- 统计数据使用阿拉伯数字：52.3分、120人、26.2%
- 均分保留一位小数：78.5分
- 百分号前不加空格：72%（非 72 %）
- 计量单位使用中文：米、平方米、千克
- 日期格式：2026年7月5日（不用 2026-07-05）

## 九、EduPaper 导出对应关系

| 格式要素 | Markdown 标记 | DOCX 导出 | LaTeX 导出 |
|---------|-------------|-----------|-----------|
| 论文标题 | `# 标题` | Heading 1 (黑体二号) | `\title{}` |
| 一级标题 | `## 一、` | Heading 2 (黑体三号) | `\section{}` |
| 二级标题 | `### （一）` | Heading 3 (黑体四号) | `\subsection{}` |
| 三级标题 | `**1. **` | Bold (宋体小四) | `\textbf{}` |
| 正文 | 普通段落 | Normal (宋体小四) | body text |
| 摘要 | `**摘要：**` | Bold prefix + normal | `\textbf{摘要：}` |
| 表格 | pipe table | Table Grid | `\begin{table}` |
| 参考文献 | `## 参考文献` + `[n]` | Heading 2 + list | `\section{}` + list |
